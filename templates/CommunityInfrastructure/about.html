What is the site?

Part 1: (currently deployed) This website is a project aimed at providing a unmet service for the tabletop miniature community. Currently artists wishing to share their painted miniatures do so on facebook groups or platforms like instagram, the problem with this system is that these images are very difficult to find, facebook is not searchable and instagram while searchable does not provide relevant enough results to be useful. This website makes it easy to search images and discover new artists.

Part 2: (in development) Currently stores and clubs for many miniature wargames run leagues for their local Communities with the admins managing league registration and matchmaking by hand. This website also provides tools for league administrators to automate the managment of their leagues.

Gallery info

Uploading images to the gallery requires an account that has been aproved by an admin, viewing images can be done without an account.

When uploading images try to be as descriptive as possible with your tags to help enable accurate image filtering.

The image colour tags are intented to match as closely as possible without becoming too specific to help enable easier searching.



Groups & Organization Info

The groups&organizations portion of the site contains functionality to view commission painters for an area(as well as view their work) along with clubs or stores


Images can be exported to Faceook or Instagram! (proof of concept implementation, Dialog with commission painters required to realize an effective implementation)
